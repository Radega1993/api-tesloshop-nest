# Teslo API
1. clonar repositorio
2. ```yarn install```
3. Copiar archivo __.env.template__ por __.env__ y poner tus datos
4. Lenvantar DB
```
docker-compose up -d
```
6. Leventar modo dev
```
yarn start:dev
```
